# 浮上するタスク

ブラウザだけで動く、シンプルな日本語タスク管理アプリです。

## 特徴

- **インストール不要** — `index.html` をブラウザで開くだけで使えます
- **オフライン対応** — データはすべてブラウザの `localStorage` に保存されます
- **自分タスク / 委任タスク** の2種類に対応
- 優先度（高・中・低）と期限を設定できます
- 期限切れタスクは翌日に自動繰り越し
- 期限の1日前・7日前にリマインドバッジが表示されます

## 使い方

1. このリポジトリをクローンまたはZIPでダウンロード
2. `index.html` をブラウザで開く
3. タスクを入力して「追加」ボタンを押す

## ステータスの流れ

**自分タスク**
```
TODO → 完了
```

**委任タスク**
```
委任中 → 返答待ち → 完了
             ↑
           再委任（メモ必須）
```

## データについて

データはブラウザの `localStorage` に保存されます。ブラウザのデータを削除するとタスクも消えるのでご注意ください。
